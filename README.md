### 开发愿景

**用于企业解决项目开发框架搭建需求,提供快速开发平台,为公司节省开发成本,让开发人员只需要关心具体业务开发**

### 快速使用

- 获取源代码

  ```shell
  git clone https://github.com/ox-data/ox-data-cloud.git
  ```

- 编译代码

  ```shell
  # 进入父级模块
  $: cd ox-data-cloud-parent
  # 编译
  $: mvn clean install -Dmaven.test.skip
  # 回到主目录
  $: cd ..
  # 编译所有模块
  $: mvn clean install -Dmaven.test.skip
  ```
